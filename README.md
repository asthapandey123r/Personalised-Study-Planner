# Personalized Study Planner

## Project Overview
This Python script helps users create a personalized study schedule based on their goals, available study hours, and subjects. It evenly distributes study time across days, making it ideal for exam preparation.

## Features
- User input for study goals
- Input for available daily study time
- Subjects selection
- Generates a daily schedule over a set duration (default: 7 days)

## Setup Instructions
1. Ensure you have Python installed (version 3.11 recommended).
2. Clone this repository to your local machine:
   ```
   git clone <repository-URL>
   ```
3. Navigate to the project folder and run:
   ```
   python study_plan_generator.py
   ```
4. Follow the on-screen prompts to input your goals, time, and subjects.
